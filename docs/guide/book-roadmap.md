## 第一版完成标准

- 每章至少包含一个真实业务案例和一个明确的架构决策；
- 涉及代码的章节必须能按文件级路径运行和验证；
- 所有高风险操作都说明权限、审批、幂等与回滚边界；
- 每章提供可复用的契约、清单、决策方法或验证方法；
- 全书通过链接、站点构建和基础内容检查；
- 第三方材料具备清晰引用和许可说明。
